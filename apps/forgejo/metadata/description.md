# Forgejo

Forgejo is a self-hosted lightweight software forge. Easy to install and low maintenance, it just does the job.

## Features

- Git repository hosting
- Issue tracking
- Pull requests and code review
- Packages registry
- Actions (CI/CD)
- Lightweight — runs comfortably on a small VPS

## After install

Visit `https://git.ivanthegeek.com` (or whatever domain you set) to complete the setup wizard.

Then configure mirrors for any GitHub repos you want to track:

**Repository → Settings → Mirror** — set source URL, mirror interval, leave credentials blank for public repos.
